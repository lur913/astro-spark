## Getting Started

### Clone the repository

```bash
git clone https://github.com/lur913/astro-spark.git 
```

### Install dependencies

```bash
cd astro-spark
pnpm install
```

### Start the development server

```bash
pnpm run dev
```

### Build for production

```bash
pnpm run build
```
